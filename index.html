Purpose:

To learn to use basic Linux commands, Vi Text Editor, Git from the command line. To create a webpage and host it using GitHub Pages.

Linux Commands:

pwd: It tells where we are located in the file system.

ls: list all files

cd : Change the directory.

dir: It lists all the directories in the current working directory.

clear: It clears up the screen.

mkdir NAME: make directory

Vi Commands:

vi FILENAME: replaces with the actual file name

i: insert mode

esc: come out of insert mode.

shift+colon: commans prompt for vi

w: write command

q: quit command

wq: write and quit command

GitHub Commands

git init: Create the repository

git add filename: add the file(s) to git in preparation for a commit

git commit -m "message": do the commit to save changes

git checkout -b branch name: make a new branch and checkout

git checkout branchname: checkout branch

git merge branchname: merge the branch
